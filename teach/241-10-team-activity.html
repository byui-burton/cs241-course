10 Teach : Team Activity - STL and Iterators

This activity is designed to be completed during class.

Overview

This activity will help you practice the syntax of iterators and the Standard Template Library (STL). In particular, this will help you prepare to handle the collection of bullets in your project.

Your assignment is to create a vector of integers and use iterators to both display the contents and remove certain items.

Instructions

Divide your teams into groups of 2-3. You should all work together on one laptop, then make sure everyone in your team gets a copy of the work that you complete.

Begin with a blank file (e.g., (ta10.cpp).

Core Requirements

  1. Prompt the user for integers, store them in a vector and display them. In particular:

    1. Declare a vector of ints.
    2. Create a function, promptNumbers, and pass your vector to it. (Don't forget to pass it by reference!)
    3. Fill in the body of the promptNumbers function to prompt the user for integers and add them to the vector until the user enters 0.
    4. Create another function displayNumbers and pass your vector to it after calling the promptNumbers function.
    5. Fill in the body of your displayNumbers function to loop through the vector and display each number that the user entered. For this step, use the square bracket operator ([]). Also, don't pass a separate variable to the function for size, but rather, use the vector's size method for your loop.
  2. Change the displayNumbers function to use an iterator to go through the items in the list and display them.

  3. Remove the odd numbers from your vector and display it again. In particular:

    1. Create another function, removeOdds, an pass your vector to it.
    2. In main, after calling your displayNumbers function, add a call to removeOdds and then call displayNumbers again so that you will display the list before and after removing the odd numbers.
    3. Fill in the body of your removeOdds function to use an iterator to go through the list and remove any number that is odd (not numbers at odd indexes, but numbers that are not divisible by 2).
    4. Be careful using the erase method. Notice that it returns an iterator to the next item in the list, so if you try to advance it again (with something like it++ you may skip items or go off the end of your list.

Test your code on the following lists:

  1. 1,2,3,4,5
  2. 1,2,3,5,6
  3. 1,3,5
  4. 2,4,6
  5. 0 (an empty list)

When you complete these requirements, show your code to your instructor to "pass it off."

Stretch Challenges

After completing the above steps. Make sure that everyone on the entire team is to this point and understands the material. Then, if you have time, move onto the following stretch challenges.

  1. Copy your file to a new one (e.g., cp ta10.cpp ta10b.cpp), then change every instance of your vector to use a linked list.
  2. Change your displayNumbers function to pass your list as const.
  3. Change your displayNumbers function to display the list backwards.

Submission

You do not need to submit your program (just make sure that everyone has a copy of it for their reference). Instead, answer the accompanying questions in the I-Learn quiz.
