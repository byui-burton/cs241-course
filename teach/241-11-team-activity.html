11 Teach : Team Activity - Sorting

This activity is designed to be completed during class.

Overview

This activity will help you practice the merge sort algorithm.

Your assignment is to read in a list of integers from a file, sort them via a merge sort and display them.

Instructions

Divide your teams into groups of 2-3. You should all work together on one laptop, then make sure everyone in your team gets a copy of the work that you complete.

Begin with the starting file: /home/cs241/ta11.cpp

This file provides methods to read words from a file and display them. Your task is to write the merge sort.

Core Requirements

  1. Create a sort function that calls itelf. Add the base case to return if the list is small enough.
  2. Add the logic to divide the array in half and to call the sort on each half.
  3. Add logic to merge the two pieces back together.

Test your code on the following files:

  1. /home/cs241/ta11/nums1.txt
  2. /home/cs241/ta11/nums2.txt
  3. /home/cs241/ta11/nums3.txt
  4. /home/cs241/ta11/nums4.txt

When you complete these requirements, show your code to your instructor to "pass it off."

Stretch Challenges

After completing the above steps. Make sure that everyone on the entire team is to this point and understands the material. Then, if you have time, move onto the following stretch challenges.

  1. Change your code to use vectors rather than arrays.
  2. Change your code to put the numbers into a built in data structure that maintains sorted order, then iterate through it to display the sorted list.
  3. Create another function that can sort lists of strings.

Submission

You do not need to submit your program (just make sure that everyone has a copy of it for their reference). Instead, answer the accompanying questions in the I-Learn quiz.
