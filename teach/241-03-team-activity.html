03 Teach : Team Activity - Parsing Streams

This activity is designed to be completed during class.

Overview

As a team, write a program that reads a list of scriptures from a file and displays them. Use the stringstream library to process and handle lines that have multiple verses ("Moroni 10:3-5" in addition to simply "Moroni 10:4").

Instructions

Divide your teams into groups of 2-3. You should all work together on one laptop, then make sure everyone in your team gets a copy of the work that you complete.

Core Requirements

  1. Begin with the file: /home/cs241/ta03.cpp that has code similar to what you wrote last week.
    • It has a Scripture struct with a book, chapter, and verse.
    • It also has display function.
  2. Prompt the user for the name of a file.
  3. Open the file, check for failures, and close it.
  4. Read the file one line at a time, and store each line in a string.
  5. At first, assume that each scripture only has one verse associated with it.
    • Use a stringstream to parse the line into a book, chapter, verse. Store it in a Scripture struct, and pass that struct to the display function.
    • Run your program on the file: /home/cs241/ta03a.txt
    • The file format is: "Alma 1 2" for "Alma 1:2".
  6. Now, modify your program so that it can handle scriptures with a verse range (i.e., a start verse and an end verse).
    • Modify the struct to include an end verse.
    • Modify the display function to display the end verse.
    • Change your code that parses the line, to read in an end verse.
    • Run your program on the file: /home/cs241/ta03b.txt
    • The file format is: "Alma 1 2 3" for "Alma 1:2-3".
  7. Finally, modify your program so that it can handle scriptures with a single verse OR a verse range.
    • Change your code that parses the line, to try to read in an end verse, detect if it failed, and if so, set the end verse to be 0.
    • Change your display to work accordingly.
    • Run your program on the file: /home/cs241/ta03c.txt
  8. Show your code to your instructor to "pass it off."

Stretch Challenges

After completing the above steps. Make sure that everyone on the entire team is to this point and understands the material. Then, if you have time, move onto the following stretch challenges.

  1. Change your code so that in addition to the above conditions, it can also handle the case of no verses at all (e.g., "Alma 31").
    • Run your program on the file: /home/cs241/ta03d.txt.
    • The file format is: "Alma 1" for "Alma 1".
  2. Change your code to detect verse ranges that are not correct, such as "20-15" and handle them appropriately.
    • Run your program on the file: /home/cs241/ta03e.txt.
    • The file format is: "Alma 1 3 2" for "Alma 1:2-3".
  3. Change your code to handle book names that begin with a number and have a space in the book name (e.g., "1 Nephi").
    • Run your program on the file: /home/cs241/ta03f.txt.
    • The file format is: "1 Nephi 2 3 4" for "1 Nephi 2:3-4".
