04 Prove : Assignment - Ecommerce Product Inventory

Overview

Your out-of-your-home online store is really taking off!

Now you realize that it's time to get a little more serious about creating a better storefront website and a system to track your orders and products. Luckily, you are a programmer!

This week, you will create our own library to handle products: to store their information, compute shipping costs, and display receipts, etc. Next week you will build on this by adding additional functionality.

Instructions

Write a C++ program according to the following:

  1. Create a class for a product that stores the name, base price, weight, and description of a product.
  2. Create a prompt method (function) that prompts for the name, base price, weight, and description, and sets these values in the current object.
  3. Create a getSalesTax method to that returns 6% of the base price.
  4. Create a getShippingCost method that returns a flat rate of $2.00 if the item is less than 5 lbs, and $2.00 + $0.10 per pound over 5 lbs.
  5. Create a getTotalPrice method that uses your other methods to return a total price.
  6. Create three display methods to display the data in each of the following formats:
    • Advertising Profile: Should display the name, base price, and description in this format:
      Finding Peace, Happiness, and Joy by Richard G. Scott - $14.49 (Elder Scott powerfully outlines the truths we need to understand and embrace in order to experience the gifts of peace, happiness, and joy.)
    • Inventory Line Item: Should display the base price, name, and weight in this format:
      $14.49 - Finding Peace, Happiness, and Joy by Richard G. Scott - 1.5 lbs
    • Receipt: Should display the name, base price, sales tax, shipping cost, and total price, in this format:
      Finding Peace, Happiness, and Joy by Richard G. Scott Price: $ 14.49 Sales tax: $ 0.87 Shipping cost: $ 2.00 Total: $ 17.36
  7. Create any other methods that you see fit to help in your program.
  8. Create a main function that creates a new Product, calls its prompt method. Then have main prompt the user for which kind of display to see (advertising profile, an inventory line item, or a receipt for this product) and call the appropriate method.

A few other guidelines / hints:

  • Make sure your program is separated into different files (one for the class definition, one for the bodies of its functions, and one for main).
  • Your prompt function should reprompt for the price if text or a number less than 0 is provided.
  • Dollar amounts should be shown to 2 decimal places, and weights should be shown to 1 decimal place. This can be done using the following code (and include the iomanip library):
    cout.setf(ios::fixed); cout.setf(ios::showpoint); cout.precision(2);
  • Don't forget to put this assignment in its own directory and start with the files provided.
  • You may notice that your "main" in this case is pretty simple and straightforward, and doesn't necessarily do anything very impressive. This is just fine, because we are more concerned with building a library (i.e., our Product class) that can be used for other projects (e.g. next week's assignment :-) ).
  • Get started early, there are a few tricky aspects to this assignment.
