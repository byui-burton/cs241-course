03 Prove : Data Structures Homework - Vector

Overview

This assignment will help you practice the basics of using Vectors in C++. It will walk you through a series of steps and then turn you to a quiz in I-Learn to report on your findings.

Instructions

Part I - Basic Vector Operations

  1. Create a vector to hold strings.
  2. Using push_back() add 4 strings to the vector: Matthew, Mark, Luke, John.
  3. Use front() and back() to display the first and last strings.
  4. Use a for loop to iterate through the list, and display each using the square bracket (i.e., v[i]) to access each item. Make sure to use "size()" as the limit for your iterator variable (i.e., i < v.size(). )

Part II - Observing Vector Growth

  1. Create a vector to hold integers.
  2. Output its size and its capacity, note anything of interest.
  3. Add 10 numbers.
  4. Output its size and its capacity, note anything of interest.
  5. Change your code to add 100 numbers.
  6. Output its size and its capacity, note anything of interest.
  7. Finally, change your code to insert 1,000 numbers, and also have it output the size and capacity after each number is added.
  8. Note how often the capacity changes relative to the size.

After completing these instructions and the preparation reading, take the I-Learn quiz. You are encouraged to work with others on the experimenting and the quiz.
