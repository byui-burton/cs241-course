06 Prove : Data Structures Homework - Stacks

Overview

This assignment will help you practice using Stacks in C++. It will walk you through a series of steps and then turn you to a quiz in I-Learn to report on your progress.

Instructions

Write a program that will read in a file containing three kinds of braces: { }, ( ), and [ ]. Your program should display either "Balanced" or "Not balanced". To be balanced, a file must contain the same number of open braces as closed braces, and they must be closed in the reverse order of how they were opened. You should ignore all whitespace.

The following are examples of balanced braces:

{ ( [ ] ) } { } [ [ [ ( ) ] ] ]

The following are examples of braces that are not balanced:

{ ( [ { [ } ] [ ] ] [

Getting Started

We will begin the program together in class. You can start with the file we create by copying it to your current directory: "cp /home/cs241/stacks06/braces.cpp ."

TestBed and Submission

An automated TestBed script is provided for you. It will test your program on 6 different files.

testBed cs241/stacks06 braces.cpp

These files should have the following results:

  • stacks01.txt - Not balanced
  • stacks02.txt - Balanced
  • stacks03.txt - Balanced
  • stacks04.txt - Not balanced
  • stacks05.txt - Not balanced
  • stacks06.txt - Balanced

You will not submit your program for grading, instead take the accompanying I-Learn quiz to report on your progress and your understand of stacks.

You are encouraged to work with others on the programming and the quiz.
