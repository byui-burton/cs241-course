10 Prove : Data Structures Homework - Trees, Sets, and Maps

Overview

This homework has a few quiz questions about the basics of Trees, Sets, and Maps. Then there are two programming exercises to complete. You are encouraged to work together on all parts of this assignment.

Programming Exercises

Complete the following simple programs before taking the quiz:

Part I - Sets

Write a program that does the following:

  1. Declare a set of strings, using the set from the standard template library.
  2. Prompt the user for a series of words until they type the word, "quit". Insert each of these words into the set.
  3. Use an iterator to display each word in the set. Note the order.
  4. Prompt the user for an additional word and display whether or not that word is in the set.

Part II - Maps

Write a program that does the following:

  1. Declare a map that has a string as a key and an int as a value (using the map class from the standard template library).
  2. Prompt the user for a series of names along with their ages, and store these in your map. Stop when the user users "quit" for the name. (You need not worry about spaces in names.)
  3. Display all the names stored in your map.
  4. Prompt the user for a name and if that person is in your map, display their age. If not, display to the user that the person is not in the map.

Submission

There is no testBed for these assignments and they should not be submitted in the Linux lab. Instead, when you are finished, take the accompanying quiz.
