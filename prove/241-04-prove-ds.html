04 Prove : Data Structures Homework - Iterators

Overview

This assignment will help you practice the basics of using Iterators in C++. It will walk you through a series of steps and then turn you to a quiz in I-Learn to report on your progress.

Instructions

Part I - In Class

  1. Create a vector to hold integers.
  2. Prompt the user for how many numbers they would like to enter.
  3. Using a for loop and push_back() prompt the user for that number of integers and add them to the vector.
  4. Using an iterator (not just the [] notation), loop through and display each number in the vector.
  5. Using an iterator (not just the [] notation), loop through the vector BACKWARDS and display each number in the vector.

Part II - Individual Work

  1. Repeat Part I but instead of having a vector of integers, create a vector of strings.
  2. Prompt for the number of strings.
  3. Loop, prompt, push_back to put that many strings in the vector.
  4. Use iterators to display each string in the vector.
  5. Use iterators to display each string in the vector, in reverse order.

Part III - A few more tricks... (work together on this if you can!)

  1. Start with your code from either Part I or Part II.
  2. Instead of using push_back() to insert your items at the end of the vector, use insert() to put each one at the front (i.e., top, start) of the vector.
  3. Leave your code in to loop through and print them out, and observe the changes in the ordering.
  4. Remove the first half of the items in the vector.
  5. Let your code loop through and print them out.

After completing these instructions and the preparation reading, take the I-Learn quiz. You are encouraged to work with others on the programming and the quiz.
