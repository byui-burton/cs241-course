12 Prove : Assignment - Spellcheck and auto-complete suggestions

Overview

Now that we have discussed several different data structures and algorithms, we can apply those to solve interesting problems.

This week, your assignment is to write a program to check the spelling of words and also suggest other auto-completions of a word.

Instructions

Write a C++ program according to the following:

  1. Prompt the user for the location of a word bank. This file is not sorted and will contain one word per line. This set of words is what you will use to determine if a word is spelled correctly and to look up additional words.
  2. Load the word bank into a data structure in memory and display the text, "Word bank loaded".
  3. Prompt the user for words until they enter "quit".
  4. For each word entered, first determine if the word is in the word bank, and if so, display the text, "Correct spelling", if not, display, "Incorrect spelling".
  5. Next, list all the words in the word bank that start with the text entered. For example, if the user entered the text "compu", your program should list words such as: computation computational, computationally, compute, computer, computers, according to what is in the word bank.

A few other guidelines / hints:

  • Make sure to store the word bank so that it is efficient to look up words.
  • You may create any classes or methods that are helpful.
  • You may use any classes or methods from the Standard Template Library (STL) that are helpful.
  • Don't copy the large word bank file to your home directory, it is large and can fill up your disk quota.

The following word bank files are available for testing:

  • /home/cs241/words_small.txt
  • /home/cs241/words_large.txt
