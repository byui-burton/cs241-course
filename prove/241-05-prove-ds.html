05 Prove : Data Structures Homework - Linked Lists

Overview

This assignment will help you practice using Linked Lists in C++. It will walk you through a series of steps and then turn you to a quiz in I-Learn to report on your progress.

Instructions

Part I - Basic Use

  1. Import the "list" library in the standard namespace.
  2. Create a list to hold integers.
  3. Prompt the user for how many numbers they would like to enter.
  4. Using a for loop and push_back() prompt the user for that number of integers and add them to the list.
  5. Using an iterator loop through and display each number in the list.
  6. Change the insertion to use push_front() instead of push_back() and repeat the iteration / display.

Part II - Repeat for strings

  1. Repeat the first part of Part I but instead of having a list of integers, create a list of strings.
  2. Prompt for the number of strings.
  3. Loop, prompt, push_back() to put that many strings in the list.
  4. Use iterators to display each string in the list.

Part III - Removing from a list... (This one is tricky... work together on this if you can!)

  1. Start with your code from Part I.
  2. Prompt the user for a series of numbers and add them to a list of integers.
  3. Iterate through the list and display them.
  4. Remove all odd numbers from the list (leaving the evens).
  5. Iterate through the remaining items and display them.

After completing these instructions, take the I-Learn quiz. You are encouraged to work with others on the programming and the quiz.
