10 Prepare : Checkpoint B - Removing from Vectors

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the syntax of removing from vectors.

Consider that you are writing a program to track your food storage inventory. Each item in your inventory has a name and an expiration date. At some point, you realize that you have a lot of food that has expired and you want to remove it from your inventory list.

Please note that this assignment requires a little more problem solving than a typical checkpoint. The purpose of this assignment is to help prepare you to handle bullets in your Skeet project.

Instructions

For this assignment, you can do all of your work in a single .cpp file for simplicity. Recognize that much of the code (regarding the the Food class, etc.) has been provided. Your task is to concentrate on adding and removing from a vector.

  1. Copy the template file /home/cs241/check10b.cpp to your working directory.
  2. Notice that a vector of Food* has been created for you.
  3. First, the provided code calls a function promptInventory. This function will prompt the user for new food items until they enter quit. These values (the name, month, and year) are stored in simple local variables for you to use.
  4. Fill in the body of the loop in the promptInventory function to create a new Food object, populate it with the correct values, and add it to the vector.
  5. When the user is done entering food, your program will call the function, displayInventory to display each item. You need to fill in the body of this function to use an iterator to go through each item in the vector and display it.
  6. After displaying the list, the provided code will prompt the user for a current month and year.
  7. It then calls the function removeExpiredItems to remove the food from the list that has expired. You need to fill in the body of this function.
  8. Implement the removeExpiredItems function to use an iterator to go through the vector and remove each Food object that has expired.
    • This task is the real trick of this assignment.
    • An item has expired if its year is less than the current year (5/2014 has expired on 1/2015) OR if its year is the same as the current year and its month is less than the current month (8/2015 has expired on 10/2015)
    • Be careful about advancing your iterator. Remember that when you call the erase function it will return an iterator that points to the next item in the list. If you were to do another ++ operation after this, you would skip one and/or go off the end of your vector.
    • Don't forget to delete the dynamically allocated objects in addition to removing them from the vector.
    • Note that a destructor has been included on the Food class to output a message when it is being deleted. This is done so that the testBed can see that you have deleted it at the correct time. (Normally, destructors would not output anything to the screen, and one would not be necessary for the Food class anyway...We will talk more about destructors in a few weeks.)
  9. After removing the expired items from your vector, the provided code will call your displayInventory function again to display the remaining items.
  10. Finally, the provided code will call a function deleteRemainingItems to delete the last of the items. This function is provided for you (you don't have to fill it in). It may be helpful to look at this function for reference.
