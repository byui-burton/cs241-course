02 Prepare : Checkpoint B - Complex Numbers

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the mechanics of structs in C++.

Instructions

Write a C++ program according to the following:

  1. Start with the skeleton of a file at: /home/cs241/check02b.cpp
  2. Create a struct for a Complex number that has a real part (a double) and an imaginary part (a double).
  3. Write a function, prompt, to prompt the user for the real and imaginary parts of complex number, and have it return a Complex struct.
  4. Write a function, display, that accepts a Complex struct and displays it in the format: "2.1 + 3.2i". You need not worry about handling zero or negatives at this point.
  5. Fill in the body for the function, addComplex, that accepts two complex numbers and returns their sum. Remember that when adding complex numbers, the real parts are added and the imaginary parts are added.

Then, the program should do the following:

  1. Declare two objects of type Complex called c1 and c2.
  2. Call your prompt function twice to fill in the values for c1 and c2.
  3. Call your addComplex function to produce the sum.
  4. Call your display function to display the sum.

Sample Output

The following is an example of output for this program:

Real: 3 Imaginary: 5 Real: 2.1 Imaginary: 6.2 The sum is: 5.1 + 11.2i

Getting Started

Start your program by copying the skeleton provided at: /home/cs241/check02b.cpp . Don't forget to include your name, and then also "Checkpoint 02b, Complex Numbers" as the assignment number. To copy the template to your current directory, you could use something like the following:

cp /home/cs241/check02b.cpp myCheck02b.cpp

Testbed

A auto-grading testbed script is provided for you to help evaluate your program. This same testbed script will be used to grade your program. It is pass/fail, so your program must pass the testbed completely for you to receive credit for this assignment. You may run the testbed as many times as you like.

Please note that it is possible to pass the testbed without using structs, but this would not help you learn, and you will not receive credit if you do not use structs.

For example:

testBed cs241/check02b myCheck02b.cpp
