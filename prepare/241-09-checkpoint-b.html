09 Prepare : Checkpoint B - Pure Virtual Functions

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the syntax of pure virtual functions.

For this checkpoint, you will create a base class for a Shape, and two derived classes for specific shapes. The base class will have a pure virtual function, getArea() that is overriden by each derived class. You will then create a certain number of new shapes, as specified by the user, put them in the same array, and loop through and display the area of each.

Instructions

For this assignment, because the classes we are going to create are so small and simplistic, and to help you focus on just the elements of inheritance and virtual functions, rather than the mechanics of creating lots of files, you will put all of your classes at the top of the main file, rather than creating separate .h and .cpp files like we would for larger, real-world classes.

  1. Copy the template file /home/cs241/check09b.cpp to your working directory.
  2. A Shape class has been provided that has a name with its getter and setter. You need to add a pure virtual function getArea(), thus it will look as follows:
    3. Shape
    -name : string
    +getName() : string
    +setName(string) : void
    +getArea() = 0 : float
  3. A Circle class has been provided with a radius, its getter/setter and a destructor. You need to add the following:
    1. Add a Circle constructor, and in it set the name variable to be "Circle".
    2. Redefine the getArea() method to return "3.14 * radius * radius".
  4. A Rectangle class has been provided with a length and width, their getters/setters and a destructor. You need to add the following:
    1. Add a Rectangle constructor, and in it set the name variable to be "Rectangle".
    2. Redefine the getArea() method to return "length * width".
  5. The provided code supplies you with a main function. It takes care of prompting the user for the correct values, and looping through. You need to add the following to main (at the places indicated in the comments):
    1. Declare a new array.
    2. Declare a new Circle, set the radius, and add it to the array.
    3. Declare a new Rectangle, set the radius, and add it to the array.
    4. Add to the loop a cout statement to display each shape in the array along with its area, in the format "Circle - 10.32"
    5. Delete each object that was dynamically allocated.
