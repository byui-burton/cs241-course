07 Prepare : Checkpoint A - Static

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the static class variables. You will start with a simple class to model the sales of Ice Cream, and then add to it a static member to represent sales tax.

Instructions

  1. Begin with a working implementation of an IceCream class. You can copy this code to your directory via the following:
    2. mkdir check07a cd check07a cp /home/cs241/check07a/* .
  2. This class has the following class diagram:
    3. IceCream
    -flavor : string
    -price : float
    +IceCream()
    +IceCream(flavor, price)
    +getTotalPrice() : float
    +prompt() : void
    +display() : void
  3. Add to this class a public static variable named salesTax.
  4. Give this variable a default value of 0.06.
  5. Change the getTotalPrice() function to include the tax in the price.
  6. A main function is provided for you in check07a.cpp. It declares two IceCream objects, displays them, asks for an alternate sales tax, and displays them again. Currently (because you hadn't written it yet), it does not actually change the sales tax value.
  7. Change main, where noted in the comments, to actually set the new sales tax value.

Sample Output

The following is an example of output for this program:

Flavor: Vanilla Price: 3.49 Flavor: Rocky Road Price: 5.49 Menu: $3.70 - Vanilla $5.82 - Rocky Road Enter alternate sales tax: 0.10 Menu: $3.84 - Vanilla $6.04 - Rocky Road

Testbed

An auto-grading testbed script is provided for you to help evaluate your program. This same testbed script will be used to grade your program. It is pass/fail, so your program must pass the testbed completely for you to receive credit for this assignment. You may run the testbed as many times as you like.

Please note that it is possible to do this assignment without using a static member variable, but this will not help you understand the concept, and you will not receive credit if you do not follow the instructions.
