12 Prepare : Checkpoint A - Simple hash function

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the syntax of a basic hash function. You will write a hash function that takes a string and determines an index to put it in an array.

Instructions

Write a function that computes an array index give a string and an array size.

For this very simple hash implementation, you should compute the index by summing the ASCII values of each character in the string, and then computing the mod (% operator).

  1. Begin by copying the template file /home/cs241/check12a.cpp to your working directory.
  2. All you need to do is implement the getHashCode function.

Sample Output

The following is an example of output for this program:

Enter a table size: 100 Enter a string: A The hash code is: 65
Enter a table size: 100 Enter a string: waffles The hash code is: 44
Enter a table size: 200 Enter a string: waffles The hash code is: 144
