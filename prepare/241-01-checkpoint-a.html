01 Prepare : Checkpoint A - C++ Basics

After completing (or while completing) the preparation material for this week, complete the following checkpoint.

Overview

This checkpoint is intended to help you review the very basics of C++ programming and ensure that you can access the Linux Lab and submit homework.

Instructions

Write a C++ program to do the following:

  1. Display the text: Hello CS 241 World
  2. Prompt the user for their first name
  3. Prompt the user for their age
  4. Display the user's name and age back to them

Sample Output

The following is an example of output for this program:

Hello CS 241 World! Please enter your first name: Emily Please enter your age: 25 Hello Emily, you are 25 years old.

Getting Started

Start your program by copying the template provided at: /home/cs241/checkpoint.cpp . Don't forget to include your name, and then also "Checkpoint 01a, C++ Basics" as the assignment number. To copy the template to your current directory, you could use something like the following:

cp /home/cs241/checkpoint.cpp myCheck01a.cpp

Testbed

A auto-grading testbed script is provided for you to help evaluate your program. This same testbed script will be used to grade your program. It is pass/fail, so your program must pass the testbed completely for you to receive credit for this assignment. You may run the testbed as many times as you like.

For example:

testBed cs241/check01a myCheck01a.cpp

Submission

After completing your assignment and ensuring that it conforms to our style guidelines, please submit it via the Linux Lab submit system.

For example:

submit myCheck01a.cpp
